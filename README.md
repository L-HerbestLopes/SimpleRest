# English

A simple **RESTful** web service, made to learn basic request mapping and project building with **C# + ASP.NET**

It is basically a "Hello World" program, it will receive the user's name through GET request and use a random greeting for each!

If you wish to run this on your computer, on Unix-like systems you can follow these steps (ensure you have git and the dotnet CLI tool)
```
git clone https://github.com/L-HerbestLopes/SimpleRest.git
cd SimpleRest
dotnet run
```
On your browser, access ``http://localhost:5216/greeting`` or ``http://localhost:5216/greeting?name=YourName``

# Português

Um simples serviço web **RESTful**, feito para aprender request mapping básico e construção de projetos com **C# + ASP.NET**

É básicamente um programa "Hello World", ele recebe o nome do usuário pela GET request e usa um cumprimento aleatório em inglês para cada!

Se você quiser rodar o programa no seu computador, para sistemas tipo Unix você pode seguir esses passos (se certifique que você possui git e a ferramenta de CLI do dotnet)
```
git clone https://github.com/L-HerbestLopes/SimpleRest.git
cd SimpleRest
dotnet run
```
No seu navegador, acesse ``http://localhost:5216/greeting`` ou ``http://localhost:5216/greeting?name=SeuNome``
